Notes

  • Data Limitations: Some fruits, such as Cranberry and Starfruit, are not included due to missing per-kg price data in the sources.
  • Variable Prices: Fruits like Durian and Eggplant have a range of sell values, so weights are provided for both minimum and maximum values.
  • Game Mechanics: The selling price can vary based on weight and mutations (e.g., a Shocked Cactus can sell for up to 700,000 Sheckles, as noted in All Crops in Grow a Garden). These calculations use base prices without mutations.
  • Currency: The game uses Sheckles, so calculations assume the user meant Sheckles instead of dollars.
